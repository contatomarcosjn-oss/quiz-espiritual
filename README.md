# Quiz Sinais Espirituais

Mini funil em React + Vite para um quiz espiritual low ticket.

## Fluxo

Landing -> Quiz -> Captura de Lead -> Resultado -> Oferta

## Como rodar

```bash
npm install
npm run dev
```

Depois acesse a URL exibida no terminal.

## Build

```bash
npm run build
```

## Observações

- Sem backend.
- O lead é salvo no `localStorage` com nome, email, resultado e data.
- O link de checkout está em `src/components/Result.jsx`.
- O quiz é uma ferramenta de reflexão e autoconhecimento, sem promessa de milagre ou diagnóstico absoluto.
